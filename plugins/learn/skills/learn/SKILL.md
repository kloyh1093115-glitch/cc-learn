# /learn — 学習室プラグイン

## いつ使うか

- `/learn` を実行したとき

---

## ワークフロー

### Step 1: 検出とモード判定

`.learn/` ディレクトリが存在するか確認する。

- **`.learn/` が存在する** → `.learn/CLAUDE.md` を読み込み → **学習モード**へ
- **`.learn/` が存在しない** → **Step 2: オンボーディング**へ

### Step 2: オンボーディング

`AskUserQuestion` で教科書ファイルのパスを確認する。

> 学習室へようこそ。教授が解説を担当します。
> 数学の教科書（線形代数・解析学・統計幾何学・最適化）のファイルパスを教えてください。

### Step 3: 学習室を自動作成

```
.learn/
├── CLAUDE.md        ← 名刺（references/claude-md-template.md から生成）
└── knowledge/       ← 応用知識の資料が育つ場所
    └── index.md     ← 学習済み知識の目次
```

`knowledge/index.md` の初期状態：

```markdown
# 学習済み知識

<!-- 資料を作るたびにここにエントリを追記する -->
<!-- フォーマット：
- [ファイル名]
  本質: [1-2文]
  前提: [依存する知識]
-->
（なし）
```

## 学習モード

### コンテキスト構築（/learn 実行時）

1. `.learn/CLAUDE.md`（教科書パスの確認）
2. `.learn/knowledge/index.md`（学習済み応用知識の目次）

数学の基盤知識は本 SKILL.md に含まれているため、追加の Read は不要。

### 2つのモード

**「教えて」モード** — 問題や概念をその場で解説。knowledge/ は更新しない。

**「資料作って」モード** — トピックの解説資料を knowledge/ に作成。次回以降の土台になる。

### モード判定

```
「教えて」「わからない」「解説して」「この問題」 → 教えてモード
「資料作って」「まとめて」「教科書に追加」「学びたい」「解けるようになりたい」 → 資料作ってモード
迷ったら聞く。
```

---

## 「教えて」モードのフロー

```
ユーザー: 〇〇を教えて / この問題を教えて

教授:
  1. 必要な知識を特定
  2. SKILL.md の基盤インデックス + knowledge/index.md を確認
  3. 前提が揃っている → 解説
     前提が基盤にある → 本質の復習を挟んで解説
     前提が基盤にも knowledge/ にもない → その旨を伝え、
     「先にこの資料を作りましょうか？」と提案
  4. 解説は必ず「本質の復習 → 本題」の順

解説の書き方：
  - 辞書ではなく教科書。箇条書きの羅列ではなく、ストーリーのある読み物にする
  - 必要な前提知識は本質を1-2文で復習してから本題に入る
  - 数学の基盤への紐づけを明示する（「これは偏微分（解析学 第1章）の応用です」）
  - 教科書の該当セクションを参照するときは CLAUDE.md のパスから Read する
```

---

## 「資料作って」モードのフロー

```
ユーザー: 〇〇の資料を作って / この問題を解けるようになりたい

教授:
  1. 目標に必要な知識の依存関係を分析
  2. knowledge/index.md を確認し、足りない前提を特定
  3. ロードマップを提示：
     「この順で資料が必要です：1. ○○ 2. △△ 3. □□（目標）」
     概要レベルの知識は最初に簡潔な概要資料を作る
  4. ユーザーの承認を得て、1 から順に資料を作成
  5. 資料の執筆はサブエージェント（執筆家）に委託してよい
  6. 各資料が完成したら knowledge/index.md にエントリを追記
  7. 全て揃ったら目標の問題に戻る
```

### 資料の書き方ルール

- **辞書ではなく教科書。** ストーリーのある読み物にする
- **接続を滑らかに。** 前提の資料が knowledge/ にあるから「私たちは〇〇を学びました」と自然に繋げられる
- **数学の基盤に帰着させる。** 全ての説明は最終的に SKILL.md の基盤知識（線形代数・解析学・統計幾何学・最適化）に繋がる
- **概要レベルから始める。** 新しい分野の最初の資料は概要を掴ませることを優先する
- **本質の復習は軽く。** 既に知っている知識は「〇〇（knowledge/xxx.md）で学んだ通り」と1-2文で触れるだけ

### サブエージェント（執筆家）

資料の執筆が長くなる場合、TaskCreate で委託する。

起動時に渡すもの：
- 資料のトピックとゴール
- SKILL.md の基盤インデックス（教授が要約して渡す）
- 依存する knowledge/ のファイルパス
- 教科書の該当セクションのパスと行番号
- 出力先（knowledge/[ファイル名].md）

執筆家のルール：
- 指定されたファイル以外を自分で読みに行かないこと
- 資料は読み物として滑らかに書くこと
- 前提知識への参照を明示すること
- 数学の基盤への紐づけを必ず含めること

## knowledge/index.md の管理

資料を作成するたびに教授が index.md にエントリを追記する。

```markdown
- machine-learning-overview.md
  本質: データからパターンを学習する枠組み。教師あり・教師なし・強化学習の3分類。
  前提: なし（概要レベル）

- supervised-learning.md
  本質: 正解データとのズレ（損失関数）を最小化する学習。勾配降下法で最適化。
  前提: machine-learning-overview.md, 偏微分（解析学 第1章）, 勾配（解析学 第2章）
```

教授は `/learn` 起動時に index.md を読むだけで「何が学習済みか」を把握できる。

---

## 肥大化への対応

- knowledge/ のファイルが増えても、教授が読むのは index.md だけ（個別資料は必要なときだけ Read）
- index.md 自体が長くなりすぎたら、分野ごとにサブインデックスに分割を検討

# 【線形代数】

## 第1章　ベクトル空間 — 構造の棲む場所

### 動機——なぜ「空間」を抽象化するのか（線形代数.md L137-L158）
本質: 矢印・関数・多項式・確率変数に共通する「足し算と定数倍」の構造を一つの枠組みで捉えるためにベクトル空間という抽象化が必要。

### ベクトル空間の定義（線形代数.md L159-L196）
本質: 8つの公理は線形結合が well-defined になるための最小条件。線形代数の全ては線形結合の上に建つ。

### 部分空間（線形代数.md L197-L233）
本質: 「線形結合について閉じた部分集合」。同次方程式 $A\mathbf{x} = \mathbf{0}$ の解全体が典型例。

### 線形独立・生成・基底（線形代数.md L234-L294）
本質: 基底は「空間を生成するが冗長でないベクトルの組」。次元はその個数で基底の選び方によらない。

### 座標（線形代数.md L295-L327）
本質: 座標は基底という「眼鏡」を通した表現に過ぎない。眼鏡を変えれば見え方は変わるが対象は不変。

### 直和（線形代数.md L328-L351）
本質: $V = W_1 \oplus W_2$ は各元が一意に分解できること。$\dim V = \dim W_1 + \dim W_2$。

## 第2章　線形写像 — 構造を保つ変換

### 線形写像の定義（線形代数.md L400-L432）
本質: 線形結合を保存する写像。基底の像を $n$ 本決めた瞬間に空間全体の像が自動的に定まる。
$T(\alpha \mathbf{u} + \beta \mathbf{v}) = \alpha T(\mathbf{u}) + \beta T(\mathbf{v})$

### 核と像（線形代数.md L433-L464）
本質: 核は「$T$ が $\mathbf{0}$ に潰すもの」の全体、像は「$T$ が到達できる範囲」。

### 次元定理（線形代数.md L465-L491）
本質: 情報の保存則。$\dim \ker T + \dim \operatorname{Im} T = \dim V$。消える分と残る分を合わせると元の自由度。

### 表現行列（線形代数.md L492-L513）
本質: 行列は線形写像の「住所」であって「正体」ではない。第 $j$ 列は $\mathbf{e}_j$ がどこへ送られるかの座標表示。

### 基底変換（線形代数.md L514-L530）
本質: 基底を変えると表現行列は $A' = Q^{-1}AP$ と変換される。適切な基底で対角行列になることもある。

### 合成と行列の積（線形代数.md L531-L548）
本質: 行列の積は写像の合成の表現行列。$(AB)_{ij} = \sum_k A_{ik}B_{kj}$ は合成を座標で書いた必然的な形。

### 線形写像の空間（線形代数.md L549-L560）
本質: 線形写像そのものをベクトルとして扱える。$V^* = \operatorname{Hom}(V, \mathbb{K})$ が双対空間。

## 第3章　内積空間 — 角度と距離の意味論

### 内積の定義（線形代数.md L612-L639）
本質: 対称性・線形性・正定値性の3条件。同じ空間に異なる内積を入れれば異なる幾何が生まれる。

### Cauchy-Schwarz の不等式（線形代数.md L640-L663）
本質: $|\langle \mathbf{u}, \mathbf{v} \rangle| \leq \|\mathbf{u}\| \|\mathbf{v}\|$。角度の定義が矛盾なく成り立つための基盤。

### 直交性と直交補空間（線形代数.md L664-L691）
本質: 内積がゼロ = 直交。内積があれば $V = W \oplus W^\perp$ と一意に分解できる。

### 直交射影と最良近似（線形代数.md L692-L718）
本質: 直交射影は最良近似の幾何学的実現。「誤差が近似先の空間に直交する」ことが最近性の条件。線形回帰・フーリエ解析・PCA全てに通底。

### Gram-Schmidt の正規直交化法（線形代数.md L719-L773）
本質: 各ステップで「既存方向の成分を引き去って新しい直交方向だけを残す」。行列表現が QR 分解。

## 第4章　行列式と固有値 — 変換の不変量

### 行列式（線形代数.md L819-L903）
本質: 線形写像が空間をどれだけ膨張・収縮させるか。$|\det A|$ が体積変化率、符号が向き、$\det A = 0$ で非可逆。
$\det(AB) = \det A \cdot \det B$

### 固有値と固有ベクトル（線形代数.md L904-L935）
本質: 変換を「各方向への独立な伸縮」に分解。$T(\mathbf{v}) = \lambda\mathbf{v}$。全固有値の積 = $\det A$、和 = $\operatorname{tr} A$。

### 固有値の計算（線形代数.md L936-L991）
本質: 特性方程式 $\det(A - \lambda I) = 0$ で決定。固有値は相似変換で不変（写像の性質であって行列の性質ではない）。

### 対角化（線形代数.md L992-L1023）
本質: 固有ベクトルを基底に取り、変換を「各方向の純粋な伸縮」で表す。$P^{-1}AP = \Lambda$。$A^k = P\Lambda^k P^{-1}$。

### スペクトル定理（線形代数.md L1024-L1063）
本質: 実対称行列は固有値が全て実数で固有ベクトルが直交し、$A = Q\Lambda Q^T$ と直交行列で対角化できる。PCA の核心。

### 正定値行列（線形代数.md L1064-L1105）
本質: 全固有値が正 ⟺ $\mathbf{x}^TA\mathbf{x} > 0$。二次形式が「お椀」型。共分散行列やヘッセ行列の性質を支配。

## 第5章　双対空間 — 空間の「裏側」

### 双対空間の定義（線形代数.md L1158-L1185）
本質: $V^*$ はベクトルに線形にスカラーを割り当てる汎関数の全体。ベクトルが「矢印」なら双対ベクトルは「等高線の族」。

### 双対基底（線形代数.md L1186-L1214）
本質: $\varepsilon^i(\mathbf{e}_j) = \delta^i_j$。$dx_i$ は「ベクトルの第 $i$ 座標を読み取る装置」。$df = \sum_i \frac{\partial f}{\partial x_i} dx_i$ は双対基底展開。

### 双対写像と転置行列（線形代数.md L1215-L1255）
本質: $A^T$ は双対写像の表現行列。$T$ が順方向なら $T^*$ は逆方向の情報の流れ。$(S \circ T)^* = T^* \circ S^*$ が逆伝播法の本質。

### 二重双対と自然同型（線形代数.md L1279-L1298）
本質: $V \cong V^{**}$ は自然な同型（基底不要）。$V \cong V^*$ には内積が必要。$df$（自然）と $\nabla f$（内積依存）の違いの根源。

### Riesz の表現定理（線形代数.md L1299-L1332）
本質: 内積があれば汎関数をベクトルで代理できるが、内積を変えれば代理ベクトルも変わる。自然勾配法の基盤。

### 逆伝播法の双対的構造（線形代数.md L1333-L1348）
本質: 逆伝播は双対写像の合成。各層の転置行列が逆方向に順に掛けられる。特異値の乖離で勾配消失・爆発。

## 第6章　行列分解 — 構造の解剖学

### SVD（線形代数.md L1397-L1479）
本質: どんな行列も「回転 → 伸縮 → 回転」に分解。$A = U\Sigma V^T$。ランク・核・像・ノルムを一挙に明らかにする。

### 低ランク近似（線形代数.md L1480-L1502）
本質: 特異値の上位 $k$ 個で打ち切る近似が最適（Eckart-Young）。PCA・画像圧縮・推薦システムの基盤。

### 擬似逆行列（線形代数.md L1503-L1532）
本質: $A^+ = V\Sigma^+ U^T$。解がない場合は最小二乗解、解が無数にある場合はノルム最小解を与える。

### QR 分解（線形代数.md L1533-L1574）
本質: Gram-Schmidt の行列表現。$A = QR$。最小二乗法を数値的に安定に解ける。

### Cholesky 分解（線形代数.md L1575-L1612）
本質: 正定値対称行列の「平方根」。$A = LL^T$。多変量正規分布のサンプリングに直結。

# 【解析学】

## 第1章　全微分の真実

### なぜ「線形写像」なのか（解析学教科書.md L169-L208）
本質: 微分は「方向から傾きへの対応」であり、ベクトルを受け取って変化量を返す線形写像。

### 線形写像としての微分（解析学教科書.md L209-L251）
本質: $f(\mathbf{a}+\mathbf{h}) = f(\mathbf{a}) + L(\mathbf{h}) + o(\|\mathbf{h}\|)$ を満たす線形写像 $L$ が全微分。微分は数ではなく写像。

### $dx$ の正体（解析学教科書.md L253-L333）
本質: $dx_i$ は「無限小量」ではなく、$\mathbf{h}$ の第 $i$ 成分を返す線形形式（双対基底）。$dx_i(\mathbf{h}) = h_i$。

### 全微分の表現（解析学教科書.md L335-L358）
本質: $df_i(\mathbf{a}) = \sum_j \frac{\partial f_i}{\partial x_j} dx_j$。偏微分係数がヤコビ行列の成分。

## 第2章　ヤコビ行列と座標変換

### ヤコビ行列（解析学教科書.md L470-L500）
本質: $J_f(\mathbf{a})$ は全微分の標準基底に関する表現行列。$df(\mathbf{a})(\mathbf{h}) = J_f(\mathbf{a})\mathbf{h}$。

### ヤコビアン（解析学教科書.md L502-L563）
本質: $\det J_\Phi$ は座標変換による向き付き体積の変化率。逆写像定理は「接空間の線形写像が可逆 → 元の写像も局所的に可逆」。

### 基底変換と表現行列の変換（解析学教科書.md L565-L582）
本質: 基底を変えると表現行列は $A' = P^{-1}AP$。写像自体は不変。

## 第3章　連鎖律の深層

### 一般形の連鎖律（解析学教科書.md L675-L736）
本質: 合成関数の全微分は線形写像の合成。$d(g \circ f)(\mathbf{a}) = dg(f(\mathbf{a})) \circ df(\mathbf{a})$。ヤコビ行列の積。

### バックプロパゲーション（解析学教科書.md L739-L763）
本質: 順伝播はデータが前へ、逆伝播は勾配が後ろへ。連鎖律が「合成の順序を逆転して計算できる」性質が数学的根拠。

### 逆写像と陰関数の微分（解析学教科書.md L765-L784）
本質: $d(f^{-1})(f(\mathbf{a})) = (df(\mathbf{a}))^{-1}$。陰関数の微分も連鎖律から導出。

## 第4章　高階微分とヘッセ行列

### 双線形写像と二次形式（解析学教科書.md L856-L893）
本質: $B(\mathbf{u}, \mathbf{v}) = \mathbf{u}^T A \mathbf{v}$。対称な場合の $Q(\mathbf{v}) = \mathbf{v}^T A \mathbf{v}$ が二次形式。

### ヘッセ行列（解析学教科書.md L895-L918）
本質: 2階偏微分を並べた対称行列。$C^2$ 級ならシュワルツの定理で対称。接空間上の対称双線形形式。

### テイラー展開と2次近似（解析学教科書.md L921-L953）
本質: $f(\mathbf{a}+\mathbf{h}) \approx f(\mathbf{a}) + df(\mathbf{a})(\mathbf{h}) + \frac{1}{2}\mathbf{h}^T H_f \mathbf{h}$。定数・接平面・曲率の3層構造。

### 極値判定（解析学教科書.md L956-L1005）
本質: 臨界点でヘッセ行列が正定値 → 極小、負定値 → 極大、不定 → 鞍点。固有値の符号で判定。

### ニュートン法（解析学教科書.md L1008-L1020）
本質: $\mathbf{x}_{k+1} = \mathbf{x}_k - H_f^{-1} \nabla f$。ヘッセ行列（曲がり具合）を考慮し2次近似の極小点へ直接ジャンプ。

## 第5章　積分の意味論と変数変換

### リーマン積分（解析学教科書.md L1093-L1120）
本質: リーマン和の極限。幾何的には面積、代数的には関数空間から実数への線形汎関数。

### 変数変換公式（解析学教科書.md L1144-L1181）
本質: $\int_D f(\mathbf{x})d\mathbf{x} = \int_E f(\Phi(\mathbf{u}))|\det J_\Phi|d\mathbf{u}$。ヤコビアンで体積要素の歪みを補正。

### 確率密度変換（解析学教科書.md L1184-L1199）
本質: $f_\mathbf{Y}(\mathbf{y}) = f_\mathbf{X}(g^{-1}(\mathbf{y})) \cdot |\det J_{g^{-1}}(\mathbf{y})|$。積分の変数変換の確率版。

# 【統計幾何学】

## 第1章　確率変数の幾何学

### 確率変数はベクトルである（統計幾何学.md L120-L159）
本質: 確率変数は $L^2$ 空間においてベクトル空間の構造を持つ。足し算・スカラー倍が自然に定義される。

### $L^2$ 内積（統計幾何学.md L183-L229）
本質: $\langle X, Y \rangle_P = \mathbb{E}[XY]$。確率分布が内積を歪める「霧」の役割を果たす。

### 共分散の幾何学（統計幾何学.md L232-L310）
本質: 共分散 = 内積、分散 = ノルムの二乗、無相関 = 直交、相関係数 = $\cos\theta$。統計学のピタゴラスの定理が直交性から従う。

### 条件付き期待値と直交射影（統計幾何学.md L330-L374）
本質: $\mathbb{E}[X \mid \mathcal{G}]$ は $X$ の可測関数空間への直交射影。「利用可能な情報による最良予測」の幾何学的正体。

### 集中不等式（統計幾何学.md L399-L429）
本質: 分散が小さい確率変数は平均周りに集中。チェビシェフ: $P(|X - \mathbb{E}[X]| \geq a) \leq \mathrm{Var}(X)/a^2$。

## 第2章　分布の変換と空間の歪み

### 確率密度変換（統計幾何学.md L475-L541）
本質: 変換で空間が伸びれば密度は薄まり、縮めば濃くなる。$n$ 次元ではヤコビアン $|\det J|$ で補正。

### 畳み込みと和の分布（統計幾何学.md L544-L576）
本質: 独立な確率変数の和の密度は畳み込み。モーメント母関数で畳み込みが積に変わる。

### デルタ法（統計幾何学.md L579-L599）
本質: 微小範囲では変換は線形。漸近分散は $[g'(\theta)]^2$ 倍に変換される。

## 第3章　情報幾何への入り口

### スコア関数（統計幾何学.md L682-L713）
本質: $s(\theta) = \partial_\theta \log f$。尤度の勾配。$\mathbb{E}[s] = 0$ なのでスコアの分散がそのまま情報量。

### フィッシャー情報量（統計幾何学.md L717-L771）
本質: スコアの分散 = 対数尤度の曲率。山が鋭いほど推定しやすく情報量が大きい。
$I(\theta) = \mathrm{Var}(s(\theta)) = -\mathbb{E}[\partial^2_\theta \log f]$

### クラメル・ラオの下界（統計幾何学.md L789-L810）
本質: 不偏推定量の分散は $1/I(\theta)$ 以下にできない。Cauchy-Schwarz の帰結。

### KLダイバージェンスとフィッシャー計量（統計幾何学.md L813-L838）
本質: フィッシャー計量はKLダイバージェンスの2次展開。大域的距離の微分が微小距離。

## 第4章　回帰分析の真実

### 正規方程式（統計幾何学.md L902-L930）
本質: 「残差が列空間に直交する」条件だけから導かれる。$\hat{\boldsymbol{\beta}} = (X^TX)^{-1}X^T\mathbf{y}$。微分不要、ピタゴラスの定理。

### 射影行列（統計幾何学.md L933-L961）
本質: $H = X(X^TX)^{-1}X^T$ は対称冪等。てこ比 $h_{ii}$ がデータ点の影響力。

### 決定係数の幾何学（統計幾何学.md L964-L988）
本質: TSS = ESS + RSS はピタゴラスの定理。$R^2 = \cos^2\theta$。

### ガウス・マルコフの定理（統計幾何学.md L992-L1009）
本質: 直交射影が最短距離を与える幾何学的事実の統計的翻訳。OLSは最良線形不偏推定量。

### 多重共線性（統計幾何学.md L1033-L1049）
本質: 列ベクトルがほぼ平行 → 列空間が「薄い」 → 射影面が不安定。条件数が爆発。

## 第5章　ベイズ推論の幾何学

### ベイズの定理（統計幾何学.md L1135-L1167）
本質: 事後分布 = 尤度 × 事前分布 / 正規化定数。精度は加算的。データ増加で事前分布の影響消失。

### 共役事前分布（統計幾何学.md L1170-L1211）
本質: 十分統計量の蓄積。正規-正規共役では精度加重平均。

### MAP推定と正則化（統計幾何学.md L1214-L1232）
本質: 一様事前 → MAP = MLE。$\log p(\theta)$ が罰則項 = 正則化の正体。

### BvM定理（統計幾何学.md L1235-L1252）
本質: 大標本で事後分布は正規分布に収束。分散はクラメル・ラオ下界に一致。ベイズと頻度主義の合流。

### ジェフリーズ事前分布（統計幾何学.md L1296-L1317）
本質: $p_J(\theta) \propto \sqrt{I(\theta)}$。統計多様体上の自然な体積要素。パラメータ変換不変。

## 第6章　主成分分析と次元削減

### 共分散行列と固有値分解（統計幾何学.md L1398-L1421）
本質: $S = P\Lambda P^T$。データの散らばりの形状を主軸座標系で表す。スペクトル定理の統計的着地点。

### レイリー商と分散最大化（統計幾何学.md L1424-L1456）
本質: $R_S(\mathbf{v})$ の最大値 = 最大固有値。逐次的に直交制約下で最大化 → 上位固有ベクトル = 主成分方向。

### SVDとの関係（統計幾何学.md L1503-L1530）
本質: 主成分方向 = 右特異ベクトル。次元削減 = 低ランク近似（Eckart-Young）。

### 確率的PCA（統計幾何学.md L1545-L1566）
本質: 「低次元潜在変数 + 等方的ノイズ」の確率モデル。欠損値・モデル選択・ベイズ拡張が可能に。

## 第7章　正則化の幾何学

### リッジ回帰（統計幾何学.md L1612-L1668）
本質: 各固有値方向に $d_i = \sigma_i^2/(\sigma_i^2 + \lambda)$ で縮小。小特異値方向（ノイズ）を抑制。
$\hat{\boldsymbol{\beta}}_{\mathrm{ridge}} = (X^TX + \lambda I)^{-1}X^T\mathbf{y}$

### ラッソ回帰（統計幾何学.md L1671-L1708）
本質: $L^1$ ペナルティで軟閾値。多くのパラメータをゼロに（自動的な特徴選択）。菱形の角にスパース解。

### バイアス・バリアンス分解（統計幾何学.md L1711-L1738）
本質: MSE = バイアス² + バリアンス。正則化はバリアンス減・バイアス増のトレードオフ。

## 第8章　確率過程の幾何学

### マルコフ連鎖（統計幾何学.md L1832-L1873）
本質: 定常分布は推移行列の固有値1の固有ベクトル。スペクトルギャップ $1 - |\lambda_2|$ が混合速度。

### MCMC法（統計幾何学.md L1877-L1907）
本質: 詳細釣り合い条件で推移行列を設計し、正規化定数を迂回して事後分布からサンプリング。

### マルチンゲール（統計幾何学.md L1911-L1941）
本質: $\mathbb{E}[M_{n+1} \mid \mathcal{F}_n] = M_n$。増分が情報に直交。分散公式はピタゴラスの定理の確率版。

### ブラウン運動（統計幾何学.md L1944-L1973）
本質: ランダムウォークの連続時間極限。連続だが至る所微分不可能。確率微積分の出発点。

## 第9章　情報理論

### エントロピー（統計幾何学.md L2024-L2076）
本質: 「平均的な驚き」。正規分布は固定された平均・分散の下で最大エントロピー分布。
$H(X) = -\sum_x p(x) \log p(x)$

### KLダイバージェンス（統計幾何学.md L2078-L2108）
本質: 交差エントロピー - エントロピー = 「間違った分布の余分な符号長」。交差エントロピー損失最小化と等価。

### 相互情報量（統計幾何学.md L2111-L2142）
本質: $I(X;Y) = H(X) - H(X \mid Y)$。$Y$ を知ることで $X$ の不確実性がどれだけ減るか。線形関係以外も捕える。

### 変分推論（統計幾何学.md L2145-L2173）
本質: ELBO最大化 = 事後分布とのKL最小化。統計多様体の部分多様体へのKL射影。

# 【最適化】

## 第1章　凸最適化と双対性の幾何学

### 勾配と方向微分（最適化.md L89-L115）
本質: 勾配は等高面の法線。Cauchy-Schwarz から最急上昇方向が勾配方向。

### ヘッセ行列と最適性条件（最適化.md L116-L136）
本質: 固有値 = 各方向の曲率。正定値 → 極小、不定 → 鞍点。

### 凸関数と凸最適化（最適化.md L138-L166）
本質: 凸なら局所的極小 = 大域的最小。非凸（深層学習）ではこの保証なし。

### ラグランジュの未定乗数法（最適化.md L168-L199）
本質: 制約面上の極値では目的関数の勾配が接空間に直交。$\nabla f = \sum \lambda_i \nabla g_i$。$\lambda_i$ は制約緩和の感度。

### KKT条件（最適化.md L202-L216）
本質: 相補性スラック $\lambda_i g_i = 0$ が核心。制約が効いているか否かの自動判定。

### ラグランジュ双対と強双対性（最適化.md L218-L249）
本質: 主問題を上から、双対問題で下から攻め、凸+スレーター条件で一致（$p^* = d^*$）。

### SVMの双対問題（最適化.md L251-L274）
本質: 双対ではデータの内積 $\langle \mathbf{x}_i, \mathbf{x}_j \rangle$ のみで記述。カーネル置換で非線形SVMに。

### 勾配降下法の収束（最適化.md L276-L302）
本質: 収束速度は条件数 $\kappa = L/\mu$ に支配。$L$-滑らか凸で $O(1/k)$、強凸で線形収束。

## 第2章　関数空間と無限次元の幾何学

### 関数空間はベクトル空間（最適化.md L336-L358）
本質: 関数にも足し算・スカラー倍。$L^p$ 空間は目的に応じた部分空間。有限次元の直観の無限次元拡張。

### ヒルベルト空間（最適化.md L360-L392）
本質: 内積 + 完備性。$\langle f, g \rangle = \int fg\,dx$。有限次元線形代数の最も自然な無限次元版。

### フーリエ級数（最適化.md L394-L417）
本質: パーセバルの等式 = ピタゴラスの定理の無限次元版。関数を直交基底で展開。

### 射影定理（最適化.md L419-L434）
本質: 無限次元でも「最近点への誤差は部分空間に直交」。表現定理の基盤。

### リースの表現定理（最適化.md L436-L450）
本質: ヒルベルト空間の任意の連続線形汎関数は内積で表せる。RKHSの再生核はここから導かれる。

### 線形作用素（最適化.md L452-L474）
本質: 行列の無限次元版。積分作用素の核関数 $K(x,y)$ が行列成分の連続版。

## 第3章　再生核ヒルベルト空間（RKHS）

### RKHSの定義（最適化.md L517-L531）
本質: 「点評価が有界」という公理。$|f(x)| \leq C_x \|f\|_\mathcal{H}$。

### 再生核の存在（最適化.md L533-L556）
本質: リースの表現定理から $f(x) = \langle f, K(\cdot, x) \rangle$ を満たす再生核 $K$ が現れる。

### 特徴写像とカーネルトリック（最適化.md L594-L618）
本質: $K(x,y) = \langle \phi(x), \phi(y) \rangle$。特徴写像を陽に計算せずカーネル値だけで高次元内積。

### 表現定理（最適化.md L658-L682）
本質: 正則化付き最適化の解は $f^* = \sum_{i=1}^n \alpha_i K(\cdot, x_i)$。無限次元がデータ数 $n$ 個に帰着。

### カーネルリッジ回帰（最適化.md L685-L710）
本質: $\boldsymbol{\alpha}^* = (\mathbf{K} + \lambda \mathbf{I})^{-1}\mathbf{y}$。リッジ回帰の非線形一般化。

## 第4章　情報幾何学

### フィッシャー情報計量（最適化.md L789-L820）
本質: $g_{ij} = \mathbb{E}[s_i s_j]$ が確率分布の空間にリーマン計量を定める。パラメトリゼーション不変。

### KLとフィッシャー計量の関係（最適化.md L842-L862）
本質: フィッシャー計量はKLの無限小版。$D_{\mathrm{KL}} \approx \frac{1}{2} d\theta^T G d\theta$。

### 指数型分布族と双対座標系（最適化.md L864-L902）
本質: 自然パラメータ $\theta$ と期待値パラメータ $\eta$ がルジャンドル変換で結ばれる。フィッシャー行列 = 対数分配関数のヘッセ。

### EMアルゴリズム（最適化.md L938-L953）
本質: e-平坦とm-平坦部分多様体への交互射影。von Neumannの交代射影法の非線形版。

### 変分推論の情報幾何学（最適化.md L955-L974）
本質: $D_{\mathrm{KL}}(q \| p)$ 最小化はm-射影（モード集中）、$D_{\mathrm{KL}}(p \| q)$ 最小化はe-射影（モーメント保存）。

## 第5章　自然勾配法

### 自然勾配の定義（最適化.md L1058-L1077）
本質: $\tilde{\nabla}L = G(\theta)^{-1}\nabla L$。「分布の空間で測った最急降下方向」。KL制約下で最効率。

### パラメトリゼーション不変性（最適化.md L1079-L1094）
本質: 自然勾配更新はパラメータの取り方に依存しない。通常の勾配降下法にはこの性質なし。

### 指数型分布族での自然勾配（最適化.md L1096-L1114）
本質: 期待値パラメータでのユークリッド勾配降下法に等しい。学習率1で一歩で最尤推定量に到達。

### TRPO・PPO（最適化.md L1171-L1199）
本質: TRPOはKL制約付き方策改善。PPOはクリッピングによる実用的近似。

### 自然勾配とニュートン法（最適化.md L1202-L1223）
本質: 対数尤度が損失の場合は一致。フィッシャー行列は常に半正定値なので自然勾配法の方が安定。
